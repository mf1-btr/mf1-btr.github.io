Software

It is important to us that our bioinformatics research also leads to robust, easy-to-use, and powerful tools for the scientific community. Our tools are available open source and we use workflow management systems like Nextflow and containers like Docker to develop reproducible research software. Have fun trying them out and we’re always happy to receive feedback and pull requests on GitHub.

Developing bioinformatics software is almost always a team effort: the tools and pipelines shown here would not be possible without collaboration. Many thanks to all our partners and the bioinformatics community!

Featured

VirusWarn

A mutation-based early warning system to prioritize concerning SARS-CoV-2 and influenza virus variants from sequencing data.

sars-cov-2 influenza nextflow
RIBAP

A comprehensive bacterial core genome annotation pipeline based on Roary and pairwise Integer Linear Programming. Calculate core genes beyond species levels.

nextflow pipeline bacteria pangenome

More

Breakfast

Fast detection of putative transmission chains in large (>1 million) sets of SARS-CoV-2 genomic sequences.

clustering sars-cov-2
poreCoV

An easy to use, fast, and robust Nextflow pipeline for SARS-CoV-2 genome reconstruction from Nanopore sequencing data. Developed together w/ the CaSe Group.

nextflow pipeline sars-cov-2
VIRify

Detection of phages and eukaryotic viruses from metagenomic and metatranscriptomic assemblies. Developed together and mainly maintained by the Microbiome Informatics team at EBI.

nextflow pipeline metagenomics viruses taxonomy
RNAflow

A portable, scalable, and parallelizable Nextflow RNA-Seq pipeline to detect differentially expressed genes, aiming for a high level of reproducibility.

nextflow pipeline rna-seq gene-expression
POCP-nf

Nextflow pipeline for the Calculation of the Percentage Of Conserved Proteins following Qin et al. 2014. but using DIAMOND instead of BLASTP.

nextflow pipeline bacteria proteins taxonomy
PoSeiDon

A pipeline for Positive Selection Detection and recombination analysis of protein-coding genes.

nextflow pipeline positive-selection recombination
PCAGO

Interactive web service that allows analysis of RNA-Seq read count data with PCA and clustering. Go to web service. Run desktop app.

pca clustering rna-seq gene-expression web-service
CLEAN

A Nextflow pipeline for decontamination of short reads, long reads and contigs.

nextflow pipeline nanopore illumina sequencing
CoVpipe2

SARS-CoV-2 genome reconstruction for Illumina data in Nextflow.

nextflow pipeline illumina sars-cov-2
